
Springboot Sample Rest Service Template
====

Un progetto di esempio per l'esposizione di servizi REST tramite il framework springboot

Il template consente di esporre uno o più servizi rest e di ottenere un JAR autoeseguibile 
in modo da poterlo facilmente installare in un container.

Viene fornito il supporto per:

 - Esposizione dei servizi in modaità rest
 - Interfaccia di navigazione dei servizi ( http://localhost:8080/swagger-ui/index.html?configUrl=/v3/api-docs/swagger-config )
 - Swagger OpenApi v3 ( http://localhost:8080/v3/api-docs )
